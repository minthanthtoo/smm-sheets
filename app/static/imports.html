Import Guide

Accepted: .xlsx · .xlsm
File Types
  • Only `.xlsx` and `.xlsm` files are processed.
  • Multiple files can be uploaded in one run.
  • `.xls` and `.csv` are not supported.
Sheet Names
  • Table: sheet name starts with "Table".
  • Daily Sales: starts with "DailySales" or includes "PGSales"/"PG Daily Sales" (excluding "Summary"/"Report").
  • Outlet List: contains "OutletList".
  • Way Plan: contains "WayPlan" or "WayPlay".
  • PJP Outlets: contains both "PJP" and "Outlet".
Region From Filename
  • LSH: `lsh` or `lashio` in filename
  • LSO: `lso` in filename
  • KT: `kt` or `keng tung` in filename
  • MLM: `mlm` or `mawlamyaing` in filename
  • MTL: `mtl`, `meiktila`, or `mhl` in filename
  • YGN: `ygn` or `yangon` in filename
  • Otherwise: `UNK`
Header requirements (what the parser looks for)
Table sheet
Product headers: `stock id`, `particular`, `ml`, `qty`, `column1`, `sales price`
Outlet headers: `no`, `ကုန်သည်`/`ဆိုင်အမည်`, `လိပ်စာ`, `township`, `sales`/`wholesales`, `way`/`car no`, `pg name`
DailySales sheet
Core: `year`, `month`, `date`, `today`, `period`, `voucherno`, `carno`, `customerid`
Outlet/Product: `ကုန်သည်အမည်`, `township`, `လိပ်စာ`, `stockid`/`stock name`, `ml`
Qty/Finance: `salespk`, `salesbot`, `liter`, `parking`, `နှုံး`, `သင့်ငွေ`, `ကော်မရှင်`
PG Daily Sales: `pg name`, `id`, `column2`, `salesbot`, `bonus`, `mmk`
Outlet List sheet
Headers: `sr`/`no`/`စဉ်`, `ကုန်သည်`/`ဆိုင်အမည်`, `township`, `type`, `way`, `ဖုန်းနံပါတ်`, `agent`, `ဘယ်သူ့ outletလဲ`, `ရင်းနှီးမှု`
Way Plan / PJP Outlets
Way Plan: `date`, `day`, `way`, `actual way name`/`route name`, `a`, `b`, `c`, `d`, `s`, `total`
PJP Outlets uses Outlet List headers and looks for "route/way" near the top.
Tip: Save spreadsheets with values (data-only). If a header row can't be detected, that sheet is skipped.
